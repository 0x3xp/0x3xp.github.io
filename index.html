Security Research

Original security research into Windows OS internals, undocumented APIs, and offensive techniques. Each entry represents hands-on investigation — not reproduced knowledge, but original findings from working with real systems at the binary level. Documentation of curiosity operating at ring-0.

R-001 2025 MALWARE-DEV

SetupAPI Callback Redirection — Process Injection via Setup Infrastructure

Original research into abusing the Windows Setup API infrastructure as a vector for process injection and code execution. This technique exploits callback mechanisms embedded in the SetupAPI — an often-overlooked surface that predates modern EDR telemetry and sits outside typical userland hook coverage.

The research documents how SetupAPI callback registration functions can be redirected to point to attacker-controlled code regions, achieving shellcode execution through a trusted Windows subsystem. This approach leverages the OS's own setup infrastructure rather than creating new threads or modifying well-monitored injection targets — making it stealthier against behavioral detection.

Findings include a fully working PoC, behavioral analysis under EDR monitoring tools, and documentation of the underlying NT structures involved.
