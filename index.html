R-001 2025 MALWARE-DEV

SetupAPI Callback Redirection — Process Injection via Setup Infrastructure

Original research into abusing the Windows Setup API infrastructure as a vector for process injection and code execution. Exploits callback mechanisms embedded in SetupAPI — a surface that predates modern EDR telemetry and sits outside typical userland hook coverage. Documents how callback registration functions can be redirected to attacker-controlled code regions, achieving shellcode execution through a trusted Windows subsystem. Includes a fully working PoC, behavioral analysis under EDR monitoring, and documentation of the underlying NT structures involved.
